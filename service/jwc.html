# 正方教务系统

# 1. 简介

正方教务系统 (opens new window)

学校的教务系统有三个镜像地址，具体请访问https://jwc.ysu.edu.cn/js/jwc_xuanke.htm (opens new window)

这三个系统的直链分别：网上选课系统（校内，仅限选课）、教务系统（外网访问） (opens new window)、教务镜像系统2（校内，非选课） (opens new window) 注意：都不可以通过直链访问

# 2. 使用

# 1. 登录系统

请选择 学生 方式登录 初始密码:

# 2 功能详情


该系统提供的功能非常丰富，但部分功能已经被其他系统所代替

  • 网上选课(废弃，已被学生选课系统 (opens new window)代替)

  • 活动报名

  • 教学质量评价(废弃，已被教学质量管理平台 (opens new window)代替)

  • 信息维护：目前看来基本没什么用

    • 个人信息
    • 密码修改

  • 教务处留言(废弃)

  • 公共信息:

  • 毕业设计

    • 查询毕设成绩与绩点

  • 信息查询

    • 学生个人课表： 与班级课表类似，只是增加了选修课程
    • 学生考试查询：
    • 等级考试查询：如英语四六级考试
    • 培养计划查询：
    • 教材使用情况查询
    • 教室查询

  1. 请使用正规浏览器从教务处网站入口访问！

    请从 教务处 (opens new window) 直接点击进入教务系统，不要使用新窗口打开

  2. 我可以通过手机使用么？ 可以单不推荐 该系统手机兼容性不太好，部分UI组件会出现重叠
